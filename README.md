## 📌 Overview

Antigravity Life OS is a personal productivity dashboard designed to act like a **second brain**.  
It helps you manage tasks, track habits, generate AI-powered schedules, and improve focus — all in one place.

---

## ✨ Features

- 📋 **Task Dashboard** – Manage your daily tasks efficiently  
- 🎯 **Active Goals Tracker** – Track progress with visual indicators  
- 🧠 **AI Planner** – Generate smart schedules automatically  
- 📚 **Study Assistant** – Analyze notes and generate summaries  
- 😊 **Mood Tracker** – Track emotional wellbeing with insights  
- 🔥 **Habit Tracker** – Build consistency with daily habits
